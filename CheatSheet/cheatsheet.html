HTML BASICS Cheatsheet
CONCEPTS SYNTAX

HEAD

* contains information about your HTML file, like its title. The title is what we see in the browser's title bar or page tab.

 
<!DOCTYPE html>
<html>
<head>
      <title>"My Webpage"</title>
    </head>
</html>

BODY

* where you put your content, such as text, images, and links. It is what will be visible on the actual page.

* paragraphs: <p> </p>

 
<!DOCTYPE html>
<html>
    <head>
        <title>"My Webpage"</title>
    </head>

    <body>
     <p>This is my first paragraph</p>
       <p>And this is the second one</p>
    </body>
</html>

<h1> The CEO </h1>
<h2> VP </h2>

<h5> Lowly assistant </h5>
<h6> Gets coffee for everyone </h6>
          
<body>
  <h1> Headline </h1>
    <p>First Paragraph</p>
    <p>Second Paragraph</p>
 </body>

HYPERLINKS

  1. First, there's an opening <a> tag and that tag has an attribute called href. The href value tells your link where you want it to go.
  2. Then you have a description of your link between your opening <a> and your closing </a> tags. This is what you will be able to click on.
  3. Finally, you have your closing </a> tag.
 
<!DOCTYPE html>
<html>
  <head>
    <title></title>
  </head>
  <body>
    <a href="http://www.codeacademy.com" > Click to go to CodeAcademy </a>
  </body>
</html>

ADDING IMAGES

* use an image tag, like so: <img>. This tag is a bit different from the others. Instead of putting the content between the tags, you tell the tag where to get the picture using src. It's also different because there is no ending tag. It has / in the tag to close it: <img src="url" />.

 
<!DOCTYPE html>
<html>
   <head>
  <title></title>
   </head>
   <body>
  <img src="http://s3.amazonaws.com/codecademy-blog/assets/f3a16fb6.jpg" />
   </body>
</html>

ADDING LINKS TO IMAGES

  1. First we open our <a> tag and point the href to URL.
  2. But this time, instead of using text inside the <a> tag, we use an <img> tag.
  3. Finally, we have our closing</a> tag.
 
<a href="http://www.codecademy.com/">
    <img src="http://s3.amazonaws.com/codecademy-blog/assets/f3a16fb6.jpg"/>
</a>

ORDERED LISTS

  1. Begin the ordered list with the opening tag<ol>.
  2. Wrap (i.e.surround) each individual item with <li> and </li> tags.
  3. Each listed item is only on one line, so put the entire element on one line.
  4. Finish the ordered list with the closing tag</ol>.
 
<!DOCTYPE html>
<html>
   <head>
  <title>Lists</title>
   </head>
   <body>
  <h2>List of things I find OK</h2>
     <ol>
        <li>Chocolate</li>
        <li>Coffee</li>
        <li>Snow</li>
     </ol>
   </body>
</html>

UNORDERED LISTS

     <ul>

NESTED LISTS

</html>

TABLES

 
<body>
    <table>

TABLE ROWS

* Use the <tr> tag to create a table row

 
<table>
       <tr></tr>
       <tr></tr>
</table>

TABLE COLUMN (DATA)

* Goes between table tags

 
<tr>
       <td> 1st Row, 1st Column </td>
       <td> 1st Row, 2nd Column </td>
</tr>
<tr>
       <td> 2nd Row, 1st Column </td>
       <td> 2nd Row, 2nd Column </td>
</tr>

TABLE BORDER

* Goes between table tags

 
<table>
    <table border="1px">
       <tr>
          <td> One </td>
       </tr>
<table>

TABLE HEADS

* Add <thead>tag above the <tbody>

* Holds the heading for each column

 
<table>
    <table border="1px">
      <thead>
         <tr>
           <th>
              Name
            </th>
            <th>
               Favorite Color
            </th>
         </tr>
       </thead>
      <tbody>
         <tr>
            <td> One </td>
         </tr>
      </tbody>
<table>

NAMING THE TABLE

* use the colspan attribute for the <th>tag

 
<table border="1px">
   <thead>
     <tr>
       <th colspan="3">3 columns across!</th>
     </tr>
   </thead>

DIVISION

* Allows you to divide your page into containers, allowing to create visual HTML objects like sidebars, menus, and more.

 
<body>
      <div style="width:50px; height:50px; background-color:red"></div>
</body>

LINK to DIVs

 
<a href="http://www.codeacademy.com">
      <div style="width:50px; height:50px; background-color:yellow"></div>
</a>

SPAN

* allows you to control styling for smaller parts of your page, such as text. For example, if you always want the first word of your paragraphs to be red, you can wrap each first word in <span> </span> tags and make them red using CSS

 
<span></span>
